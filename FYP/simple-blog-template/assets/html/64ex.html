總結

  1. 文字 → ASCII 碼 → 8位二進制
  2. 合併二進制 → 分割成6位一組
  3. 6位二進制 → 十進制 → Base64 字符
  4. 必要時添加 '=' 填充
提示： 實際應用時，建議使用 Python 的 base64.b64encode() 或在線工具，手動編碼僅用於學習！
